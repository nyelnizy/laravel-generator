# laravel-generator

## Description

A fork of laravel infyom generator


## Install
Add the following to composer.json file
```bash
    "repositories": [{
            "type": "vcs",
            "url": "git@github.com:nyelnizy/laravel-generator.git"
        }
    ],
```
Add the dependency to require block
```bash
  "infyomlabs/laravel-generator": "dev-fix-1.1"
```
Run
```bash
  composer update
```
